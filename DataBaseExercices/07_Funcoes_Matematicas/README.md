# Lista de Exercícios de Programação

1. **Calcular volume e área de um cilindro**  
   Codifique um programa para calcular o volume e a área de um cilindro com raio de 6 cm e altura de 5 cm.

2. **Calcular a hipotenusa de um triângulo retângulo**  
   Desenvolva um programa que calcule a hipotenusa de um triângulo retângulo. O usuário deve inserir os valores dos catetos.  
   **Fórmula da hipotenusa:**  
   $h = sqrt(cateto1^2 + cateto2^2)$

3. **Exibir lados de um triângulo retângulo a partir de hipotenusa e ângulo**  
   Crie um programa que solicita a hipotenusa (h) e o ângulo α de um triângulo retângulo. Em seguida, exiba os valores dos três lados desse triângulo.  
   **Dica:**  
   - Cateto adjacente: $c1 = cos(α) × h$
   - Cateto oposto: $c2 = sin(α) × h$  
   - Hipotenusa: $h = sqrt(c1^2 + c2^2)$

4. **Número aleatório a partir de uma semente**  
   Desenvolva um programa que recebe um número inteiro (n) e imprime na tela um número aleatório usando n como semente.  
   **Dica:** Pesquise sobre a função `rand`.

5. **Calcular distância entre dois pontos**  
   Escreva um programa que calcule a distância entre dois pontos (x1, y1) e (x2, y2).  
   **Observação:** Todos os números e valores de entrada/saída devem ser do tipo real.  
   **Fórmula da distância:**  

   $dist = sqrt((x1 - x2)^2 + (y1 - y2)^2)$
