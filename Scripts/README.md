# Tarefa de Auditoria

**Objetivo:**  

Criar um conjunto de scripts de auditoria (`*.sh`) para:

- Contar quantos pacotes estão instalados no sistema;
- Contar quantos arquivos existem em um diretório `X`;
- Verificar o gerenciamento de volumes;
- Verificar a latência;
- Monitorar logins;
- Verificar serviços: Apache, MySQL e SSH;
  
**Observação:**  
- Enviar os resultados por e-mail.

**Observações adicionais:**

- Utilize comandos básicos do Linux, o comando `mail` e expressões regulares (Regex) quando necessário.
- Certifique-se de ter permissões adequadas para executar os scripts.
- O serviço de e-mail deve estar configurado no sistema, conforme o mapa de rede.
